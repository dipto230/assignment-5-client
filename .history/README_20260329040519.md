## 📌 Project Overview

LawHive is a modern legal service platform where users can find and book lawyers based on their needs.

This frontend application provides:

* Clean and responsive UI
* Role-based dashboards (User, Lawyer, Admin)
* Booking system with schedule management
* Secure authentication flow

---

## 🔗 Live URL

* 🌍 Frontend: https://assignment-5-client-six.vercel.app

---

## 🚀 Features

### 👤 User

* Browse lawyers by category
* View lawyer details and expertise
* Book appointments based on availability

### ⚖️ Lawyer

* Manage personal schedule
* Set availability
* View and manage bookings

### 🛠️ Admin

* Manage lawyer categories
* Control platform structure
* Manage users and lawyers

## ⚙️ Setup Instructions

### 📥 Clone Repository

```bash
git clone https://github.com/your-username/lawhive-frontend.git
cd lawhive-frontend
```

---

### 📦 Install Dependencies

```bash
pnpm install
```

or

```bash
bun install
```

---

### 🔑 Environment Variables

```env
NEXT_PUBLIC_API_BASE_URL=https://assignment-5-backend-sepia.vercel.app/api/v1
JWT_ACCESS_SECRET=youraccesstokensecret
```

---

### ▶️ Run Development Server

```bash
pnpm dev
```

or

```bash
bun --bun next dev
```
